Widget Profiles you can save and manage widget setups and arrangements. With a built in Undo feature you have full control of the design of your 
site.

== Description ==

Well this was one of my smaller ideas in my Google Summer of Code application, which I.m guessing i didn.t get, but i figured i would release it 
regardless.  So to the point.

Have you ever been playing around with your widgets and or new themes, but didn.t want to loose your current widget placement. Well with Widget 
Profiles you can save and manage a multitude of widget setups and arrangements. With a built in Undo feature you finally have full control of the 
design of your site.

*   Save Widget Profiles
*   Manage Unlimited Widget Profiles
*   Dummy Proof Undo Feature

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `widgetprofiles.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Look under the 'Design' menu for 'Widget Profiles' to manage your profiles.
